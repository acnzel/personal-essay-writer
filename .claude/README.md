# 📝 Personal Essay Writer

Claude와 함께 성장하는 글쓰기 시스템입니다.

## 🎯 목적

개인적인 견해, 사람에 대한 이해, 삶의 지혜와 가치관을 **나만의 목소리**로 글을 쓰는 것을 돕습니다.

## 🏗️ 구조

```
.claude/
├── agents/                          # 4개의 협업 에이전트
│   ├── style-analyzer.md            # 글 스타일 분석
│   ├── content-writer.md            # 글 작성
│   ├── content-reviewer.md          # 글 검토 (80점 미만 시 수정 요청)
│   └── style-learner.md             # 피드백 학습
│
└── skills/
    └── personal-essay/
        ├── SKILL.md                 # 메인 오케스트레이터
        └── data/
            ├── style-guide.md       # 스타일 가이드 (핵심!)
            ├── feedback-log.md      # 피드백 기록
            ├── samples/             # 참고 글 원본
            ├── approved-posts/      # 승인된 글
            └── style-history/       # 가이드 버전 이력
```

## 🔄 워크플로우

```
Writer가 글 작성 → Reviewer가 검토 → 통과하면 보여줌
                                    → 제 피드백을 받으면
Style Learner가 학습 → Analyzer가 가이드 업데이트
```

## 📱 사용법

### Claude 모바일/데스크톱 앱에서

1. 이 저장소를 프로젝트로 연결
2. "글 써줘 [주제]" 로 글 작성 요청
3. 결과물에 피드백 ("좋아", "이건 좀 아닌데")
4. 시스템이 학습하며 점점 나다워짐

### 명령어

| 명령 | 설명 |
|------|------|
| `글 써줘 [주제]` | 새 글 작성 |
| `샘플 추가해줘` | 참고할 내 글 추가 |
| `스타일 가이드 보여줘` | 현재 학습된 스타일 확인 |
| `피드백 기록 보여줘` | 지금까지의 피드백 확인 |

## 🌱 성장 구조

| 단계 | 상태 | 특징 |
|------|------|------|
| 초기 | 🌱 | 기본 스타일 가이드로 시작 |
| 성장 | 🌿 | 피드백이 쌓이며 정교해짐 |
| 성숙 | 🌳 | 나의 목소리를 정확히 재현 |
